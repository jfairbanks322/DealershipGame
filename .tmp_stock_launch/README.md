# GOLD MARKET

A classroom prediction market simulator designed to teach:

- how shared odds move when students trade into the same pool
- why bettor identities should stay private even when the line is public
- how hype, FOMO, and chasing losses can turn forecasting into gambling
- why staying in cash is sometimes the healthiest decision

## Run it

```bash
npm start
```

Then open [http://localhost:3000](http://localhost:3000).

## Default teacher login

- Username: `teacher`
- Password: `market-open`

## Launch notes

- The old Stock Arena experience has been replaced at the same address with the GOLD MARKET dashboard.
- Student registration supports saved avatars.
- Teacher controls can publish custom markets on the fly, resolve them `YES` or `NO`, archive them, and refill wallet floors.
- The live board uses 5-share trade blocks with a shared class pool and private student positions.
- Students can earn steady money through daily off-market tasks even if they stay out of the market.
